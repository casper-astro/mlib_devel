Description

A freeze counter is an enabled counter which holds its final value (regardless of enables) until it is reset. Thus, a freeze counter will count from 0 to 31 on 31 enables, but will hold 31 thereafter until a reset occurs. This block is useful for writing data in a single pass to memory without looping.
